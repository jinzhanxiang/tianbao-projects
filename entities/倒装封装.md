# 倒装封装

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

技术，通过将芯片颠倒封装在基板上，芯片与外部系统通过焊球或凸块实现链接，封装更为紧凑。倒装封装在PAD上沉积锡铅球，然后将芯片翻转加热，利用熔融锡铅球与陶瓷基板相结合。东吴证券研究所指出，与传统引线键合相比，倒装封装在封装密度和处理速度上显著更高。
