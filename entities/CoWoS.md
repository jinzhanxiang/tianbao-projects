# CoWoS

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

芯片基板上芯片封装技术（Chip on Wafer on Substrate），英伟达H100由1颗Hopper GPU+8颗HBM通过CoWoS技术封装集成，实现GPU与HBM的高速互联。HBM堆叠完成后以KGD形式与处理器芯片集成到CoWoS结构中。CoWoS结构测试需要平衡高质量与低成本，涉及HBM堆栈、信号处理芯片、中介层芯片的协同测试

### 技术维度
- 根据数据，2010年代芯片仅需进行三种类型测试，而2020年代新增了新故障模型测试、多裸片测试等环节，测试流程从2010年代的简单五步演变为2020年代的复杂多步流程。扫描测试虽能通过一个管脚测试多个 (置信度: 0.9500)
- 对比传统DRAM测试流程：仅需DRAM晶圆测试和封装级测试。HBM测试流程新增逻辑芯片测试、KGD测试（老化、高低温、TSV、Pins、CoWoS），且由于超过10000个Pins受封装限制只能通过A (置信度: 0.9500)
- 2.5D封装的典型应用是台积电的CoWoS，需要硅中介层提供高密度布线平面；3D封装的典型应用是SK海力士、三星的HBM，直接将存储芯片垂直堆叠。两种架构的核心差异在于是否使用中介层，以及TSV的布线 (置信度: 0.9300)
- 基于CoWoS封装技术是一种2.5D技术，通过在GPU和HBM之,推导出CoWoS和3D封装技术需要先进的封装设备支撑，包括减薄机、 (置信度: 0.9000)
- 通过对比2.5D TSV需要中介层实现芯片与基板互联，典型应用为台积,得出2.5D TSV适合逻辑芯片与高带宽存储的集成（如AI芯片） (置信度: 0.9000)
