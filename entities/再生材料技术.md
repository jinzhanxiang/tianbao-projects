# 再生材料技术

**类型:** TECH (技术)
**行业:** 新能源电池行业
**核心实体:** ✅

再生材料技术涵盖正极材料再生（锂、钴、镍回收）、负极材料再生、电解液处理、隔膜回收等全组分回收工艺。针对磷铁渣等附加值较低材料，需重点突破高效全组分回收技术，推动资源利用最大化。再生黑粉进口已对原生矿产供应产生影响。
