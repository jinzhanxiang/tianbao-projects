# GAA晶体管架构

**类型:** TECH (技术)
**行业:** 光伏设备

架构FinFET自2012年导入以来，支撑至5nm节点，GAA预计2025年量产2nm，并在1.4nm/1nm节点持续演进，CFET有望于2031年以后导入至7Å以下工艺节点，配合背面供电、EUV等先进工艺同步升级。GAA晶体管结构由横向变为环绕甚至垂直堆叠，带来栅极结构、金属材料、隔离层等层级的大幅增加。
