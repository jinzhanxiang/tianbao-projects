# 高带宽存储器HBM

**类型:** PRODUCT (产品)
**行业:** 半导体
**核心实体:** ✅

高带宽存储器HBM" (High Bandwidth Memory HBM), , within 其高带宽（≥1 TB/s）和低功耗特性，使其成为人工智能、深度学习加速器以及高性能计算（HPC）系统的首选存储方案。随着 5nm/3nm 工艺和 CoWoS、SolC 等先进封装技术的成熟，HBM 在服务器、GPU、AI 加速卡和汽车电子等高端应用中的渗透率快速提升，已成为半导体存储产业向高密度、低功耗转型的关键推动。
