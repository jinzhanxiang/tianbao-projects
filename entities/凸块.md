# 凸块

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

在芯片上制作金属凸块提供电气互连的技术，是先进封装四要素之一。根据材质分为金凸块、铜镍金凸块、铜柱凸块、锡凸块。金凸块主要应用于显示驱动芯片、传感器等产品；铜柱凸块广泛应用于通用处理器、存储器芯片、AI芯片等领域。东吴证券研究所指出，凸块制作材质主要有金、铜、铜镍金、锡等，不同金属材质适用于不同芯片封装
