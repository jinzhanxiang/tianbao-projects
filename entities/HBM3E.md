# HBM3E

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

海力士开发的下一代高带宽存储器，H200首次搭载HBM3E。HBM3E使单颗内存容量从80GB提升至141GB，带宽从3.35TB/s提升至4.8TB/s，大幅超越HBM3性能。HBM3E加持下Llama-70B推理性能几乎翻倍，GPT3-175B推理性能提升60%，是AI芯片性能提升的核心支撑技术。
