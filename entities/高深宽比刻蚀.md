# 高深宽比刻蚀

**类型:** TECH (技术)
**行业:** 光伏设备
**核心实体:** ✅

先进制程刚需技术，3D NAND通孔结构需在百纳米级孔径下实现10微米级深度，深宽比从30:1提升至60:1、90:1以上，支持堆叠层数从128层迈向300层甚至1000层。三星制造V4-64层3D NAND时HARC成本占21%，第10代开始增至35%。
