# 内存墙

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

内存墙问题包括内存容量有限、内存传输带宽不足及传输延迟问题，具体表现在计算逻辑与片上内存之间、计算逻辑与DRAM内存之间、不同sockets上不同处理器之间都存在同样的瓶颈。当模型超过单芯片可用内存容量和带宽时，需要利用分布式内存并行技术将训练/服务扩展到多个加速器，由于片间带宽小于片内带宽，会面临更严重的内存墙问题。DRAM与互联带宽增速远低于服务器硬件算力增速
