# 硫化物电解质（TECH）

**类型:** TECH (技术)
**行业:** 电力设备
**核心实体:** ✅

硫化物电解质是全固态电池的核心增量环节之一，具有高离子电导率和良好的机械柔性。从第一性原理出发，硫化物电解质配合干法电极技术能实现厚电极、减少能耗、提高压实密度、减少副反应。当前降本和量产的主要瓶颈在于硫化锂环节，因其成本高昂且工艺放大困难。当升科技、国瓷材料、容百科技等企业已布局硫化物电解质。
