# 长川科技

**类型:** ORG (组织)
**行业:** 半导体
**核心实体:** ✅

长川科技聚焦集成电路后道测试，提供测试机、分选机及探针台全套方案。通过收购新加坡STI和马来西亚EXIS，切入AOI光学检测领域并补全转塔式分选机产品线，构建完整产品矩阵与技术壁垒。自主研发的D9000系列SoC测试机已进入国内封测龙头企业供应链，实现高端测试设备关键突破。作为国产半导体测试设备主要供应商，公司在推动行业自主可控及提升国内封测产业链竞争力方面发挥重要作用
