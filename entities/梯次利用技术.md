# 梯次利用技术

**类型:** TECH (技术)
**行业:** 新能源电池行业
**核心实体:** ✅

梯次利用是指将退役动力电池应用于储能、低速电动车等对能量密度较低的场景。梯次利用技术可延长电池使用寿命5-10年，提升资源综合利用效率。目前行业正研究制定退役动力电池梯次利用与再制造的技术标准、产品认证和安全监管规范。
