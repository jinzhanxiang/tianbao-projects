# 再分布层技术

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

技术，在晶圆表面沉积金属层和相应介质层并形成金属布线，对I/O端口进行重新布局到更宽松区域并形成面阵列排布。晶圆级封装中RDL在2.5D封装中通过RDL将网络互联并分布到不同位置，将硅基板上方的Bump和基板下方的Bump连接

### 技术维度
- 传统封装采用引线方式连接芯片与外部系统，而先进封装四要素中RDL提供平面内线路重新分布，TSV提供垂直方向穿孔连接，凸块提供芯片与基底的点对点连接，三者协同实现3D堆叠封装，传输速度显著快于引线方式 (置信度: 0.9500)
