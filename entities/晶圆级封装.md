# 晶圆级封装

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

技术，在芯片还在晶圆上时就进行封装，保护层可黏接在晶圆顶部或底部，然后连接电路再将晶圆切成单个芯片。晶圆级封装分为扇入型（Fan-in WLP）和扇出型（Fan-out WLP），扇出型引脚数多于扇入型。WLP已广泛用于闪速存储器、EEPROM、高速DRAM、SRAM、LCD驱动器、射频器件、逻辑器件、电源管理器件和模拟器件等领域

### 行业维度

- 基于扇入型将导线和锡球固定在晶圆顶部，封装尺寸受芯片尺寸限制，引,推导出随着芯片功能增加和封装密度要求提升，扇出型晶圆级封装市场占比 (置信度: 0.8200)
### 技术维度
- RDL的核心价值在于突破芯片原始I/O布局的限制。芯片设计时I/O端口一般分布在边沿或四周，这对于引线键合工艺方便，但对倒装技术困难。RDL通过金属布线和面阵列排布，解决了这一矛盾，是实现2.5D/3 (置信度: 0.9400)
- 2.5D封装的典型应用是台积电的CoWoS，需要硅中介层提供高密度布线平面；3D封装的典型应用是SK海力士、三星的HBM，直接将存储芯片垂直堆叠。两种架构的核心差异在于是否使用中介层，以及TSV的布线 (置信度: 0.9300)
