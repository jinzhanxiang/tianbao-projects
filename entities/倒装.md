# 倒装

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

倒装封装（FC，Flip Chip）通过将芯片颠倒封装在基板上，芯片与外部系统主要通过焊球或凸块实现链接，封装更为紧凑。倒装封装在IOpad上沉积锡铅球，然后将芯片翻转加热，利用熔融的锡铅球与陶瓷基板相结合。当前主要应用于CPU、GPU及芯片组等产品，CPU及内存条等电子产品是最常见的应用倒装芯片技术的器件。与传统引线键合相比，倒装封装密度和处理速度显著高于引线键合技术。
