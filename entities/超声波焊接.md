# 超声波焊接

**类型:** TECH (技术)
**行业:** 电力设备
**核心实体:** ✅

固态电池专用焊接技术，原理为通过高频振动摩擦生热（频率15-40kHz），实现材料分子层面的结合，焊接周期仅需0.3秒，能耗较激光焊接降低70%，属于冷焊接避免高温破坏电极材料，可焊接铜铝异种金属、超薄箔材低至0.05mm。
