# RDL重布线层技术

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

ReDistribution Layer，先进封装中实现芯片引脚重新布局的关键技术。RDL工艺需要涂胶显影机定义RDL图形轮廓，光刻机将图形从掩膜转移到芯片表面，刻蚀机去除材料形成RDL布线结构，溅射台沉积阻挡层和籽晶层，电镀设备进行铜填充和增厚。RDL重布线层技术使芯片可使用不同位置或类型的引脚进行互联。
