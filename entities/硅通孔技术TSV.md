# 硅通孔技术TSV

**类型:** TECH (技术)
**行业:** 半导体
**核心实体:** ✅

硅通孔技术（TSV）在芯片内部垂直穿透硅片并连接芯片顶部和底部的金属线，实现不同芯片层级之间的电信号连接。TSV制作主要包括六个关键工艺步骤：深反应离子刻蚀或激光打孔制作TSV、热氧化或等离子体增强化学气相沉积形成绝缘层、物理气相沉积制作阻挡层和种子层、电镀铜填充、化学机械抛光去除多余铜、TSV Cue外露。TSV可分为2.5D和3D两种类型。
