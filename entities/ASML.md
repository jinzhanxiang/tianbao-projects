# ASML

**类型:** ORG (组织)
**行业:** 光伏设备
**核心实体:** ✅

总部位于荷兰的全球光刻机龙头，2024年全球市场份额61.2%，在EUV领域占据绝对主导地位。2023-2025年推出的EXE:5000/EXE:5200B机型采用13.5nm EUV光源、0.55NA实现8nm分辨率。NXT系列覆盖中端（ArFi光源、38nm分辨率）与成熟制程（KrF光源、110nm分辨率）。2023年9月起荷兰政府禁止公布对华销售情况，EUV光刻机被完全禁止对华出口。
