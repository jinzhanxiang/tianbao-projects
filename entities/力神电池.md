# 力神电池

**类型:** ORG (组织)
**行业:** 新能源电池行业

力神电池是锂离子电池研发与制造企业，深耕新能源动力及储能领域，具备从材料、电芯到模组的全产业链整合能力。在新兴的电动垂直起降飞行器（eVTOL）赛道，公司展现出显著的技术领先优势，其专用电池能量密度高达325Wh/kg，同时消费级无人机电池能量密度达到340Wh/kg，处于行业第一梯队。凭借在高比能电池领域的深厚积累，力神电池已成为低空经济产业链中的关键供应商，为航空级动力电池提供高安全、长寿命的解决方案，对推动eVTOL商业化落地。
