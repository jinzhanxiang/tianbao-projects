# 再生黑粉技术

**类型:** TECH (技术)
**行业:** 新能源电池行业

GB/T 45203-2024《锂离子电池用再生黑粉》国家标准对三元黑粉镍钴≥25%、锂≥3.5%，水溶性氟≤0.4%；铁锂黑粉锂≥2%、铁≥18%，水溶性氟≤0.1%，杂质铅镉铬砷均≤0.01%。海外产品因分选技术不足难以达标。
