
# 2026-07-20 main 教训汇总（主公多轮终极指令）

## 教训 A（按主公 08:23+10:11）: fetch 注释 ≠ 删除
- 错误做法: 只在 JS 里注释掉 fetch 调用，但保留 `await response.json()`
- 正确做法: 注释 fetch 后必须把 data 改为内嵌静态 const data = {...}
- 验证: `await response.json()` 出现 0 次 + `fetch('data/` 出现 0 次
- 应用: commit `3fa0ba9` 已修复 10 HTML 文件

## 教训 B（按主公 08:23+10:15）: html is not defined ReferenceError
- 错误做法: 移除 data 块时同时误删了 `let html = '';` 和 `const list = ...` 声明
- 正确做法: 任何修改 JS 块时，必须保留所有顶层变量声明
- 验证: JS 语法 `{=26, }=26` balance + Playwright DOM 真实数据
- 应用: commit `064d0a3` 已修复 reports.html

## 教训 C（按主公 08:42）: particles.js 在 <head> 中加载导致 TypeError
- 错误做法: <head> 中加载 particles.js → document.body == null → insertBefore 抛错
- 正确做法: <script src='particles.js'> 必须移到 </body> 前一行
- 验证: console_errors 8/8 = 0（修复前 8/8 = 1）
- 应用: commit `3a5b775` 已修复 8 HTML 文件
